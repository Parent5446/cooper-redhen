parse_string(self, contents)

source code 

Parse a string of JCAMP file data and extract all needed data.

Search a JCAMP file for the chemical's name, type, and spectrum data. Then integrate the X, Y data and store alGet a specific data label from the file.l variables in the object.

Parameters:
  • contents (unicode or str) - String containing spectrum information

Warning: Does not handle Windows-format line breaks.

source code 

Find spectra similar to the given one.

Find spectra that may represent the given Spectrum object by sorting the database using different heuristics, having them vote, and returning only the spectra deemed similar to the given spectrum.

Returns: list of backend.Spectrum
